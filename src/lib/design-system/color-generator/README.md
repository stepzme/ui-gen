# Color Generator

Эта папка содержит утилиты для генерации цветовых систем, которые пока не используются в приложении, но могут быть полезны в будущем.

## Файлы:

- **`colors-config.ts`** - Базовые OKLCH значения для всех цветов
- **`generate-color-scale.ts`** - Утилиты для генерации цветовых палитр (50-900)
- **`semantic-tokens-config.ts`** - Конфигурация семантических цветовых токенов

## Возможное использование:

1. **Автоматическая генерация CSS** - создать скрипт, который генерирует `globals.css` из этих конфигов
2. **Динамические темы** - использовать для создания кастомных цветовых схем
3. **Цветовые палитры** - генерировать палитры для дизайн-системы

## Текущее состояние:

Эти файлы **НЕ импортируются** в приложении. Активные цветовые переменные находятся в:
- `src/app/colors.css` - готовые CSS переменные
- `src/app/globals.css` - интеграция с Tailwind
