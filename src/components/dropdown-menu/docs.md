# Dropdown-menu Component

A dropdown-menu component for the design system.

## Usage

```tsx
import { Dropdown-menu } from "@/components/dropdown-menu/dropdown-menu"

<Dropdown-menu>
  Content
</Dropdown-menu>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |

## Examples

### Basic Usage
```tsx
<Dropdown-menu>
  Basic dropdown-menu
</Dropdown-menu>
```

## Accessibility

- Proper ARIA attributes
- Keyboard navigation support
- Screen reader compatibility
